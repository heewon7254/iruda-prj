버튼

  • - icon: 아이콘 요소 (선택사항)
  • - label: 텍스트 (선택사항)

포커스

  • - `focus-visible`은 사용자가 키보드로 포커스를 이동할 때만 표시됩니다.
  • - 마우스로 클릭하면 `outline`이 표시되지 않으며, 키보드로 포커스를 이동한 경우에만 `outline`이 보이게 됩니다.
  • - 이 스타일은 사용자의 접근성을 돕기 위해 포커스를 시각적으로 표시합니다.
&:focus-visible { outline: 1px solid black; }
